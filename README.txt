- To compile the program
  - type 'make'
- to run the program
  - ./convolve filterName.filt imageName.img [optionalOutputImage.img]

- Keyboard functions:
  - r or R will reload the original image
  - w or W will write the image to the specified output file, if one was not
    specified at run time, you will be prompted by the program to enter an output
    file name
  - f or F will prompt the user to enter the name of the new filter to load
  - q or Q will quit the program
  - i or I will invert the image colors
  - c or C will convolve the displayed image with the specified filter
